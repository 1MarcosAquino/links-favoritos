# 🔗 links-favoritos

    Uma aplicação web simples e eficiente para gerenciar seus links preferidos, utilizando o armazenamento local do navegador (IndexedDB) para garantir que seus dados persistam sem a necessidade de um banco de dados externo.

## 🚀 Funcionalidades

    Listagem Dinâmica: Visualização clara dos links salvos usando DataTables.

    CRUD Completo: Adicione, edite e remova links com facilidade.

    Interface Responsiva: Design adaptável para diferentes tamanhos de tela com Bootstrap.

    Experiência do Usuário: Modais elegantes e alertas de confirmação para ações críticas.

    Persistência Local: Os dados ficam salvos no seu navegador através da API IndexedDB.

## 🛠️ Tecnologias Utilizadas

    O projeto foi construído com as seguintes ferramentas:

| Tecnologia  | Descrição                                      |
| ----------- | ---------------------------------------------- |
| jQuery      | Manipulação do DOM e eventos.                  |
| Bootstrap   | Estruturação visual e componentes responsivos. |
| SweetAlert2 | Modais customizados para formulários e avisos. |
| DataTables  | Gerenciamento e formatação da tabela de dados. |
| IndexedDB   | Banco de dados NoSQL embarcado no navegador.   |
